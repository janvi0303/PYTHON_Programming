1. Create Virtual Environment

Open CMD inside the project folder and run:

python -m venv venv

2. Activate Virtual Environment
Windows
venv\Scripts\activate


(You should now see (venv) before your command prompt.)

3. Install Required Libraries

Run the following inside the activated venv:

pip install pyttsx3 PyPDF2


Tkinter comes pre-installed with Python, no need to install it.

4. Run the Application

Execute the Python file using:

python python_dev_task3.py


This will launch the GUI.

5. Deactivate the Virtual Environment (Optional)

When you are finished:

deactivate
